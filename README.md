# Банковское приложение

Консольное приложение на языке Go, реализующее базовые банковские операции: создание счета, пополнение, снятие, перевод между счетами, просмотр баланса и истории транзакций.

## Функциональные возможности

- **Создать счет** – позволяет пользователю создать новый банковский счёт, указав имя владельца.
- **Пополнить счет** – добавляет указанную сумму на счёт.
- **Снять средства** – списывает указанную сумму со счёта.
- **Перевести другому счёту** – переводит средства на другой счёт по его ID.
- **Просмотреть баланс** – отображает текущий баланс счёта.
- **Получить выписку** – показывает историю всех транзакций по счёту.
- **Выход** – завершает работу приложения.

## Технические особенности

- Программа использует два основных интерфейса:
  - `AccountService` – для работы с банковским счётом.
  - `Storage` – для инкапсуляции логики хранения данных.
- Реализована система кастомных ошибок:
  - `ErrInsufficientFunds`
  - `ErrInvalidAmount`
  - `ErrAccountNotFound`
  - `ErrSameAccountTransfer`
- Хранилище реализовано в памяти (`InMemoryStorage`) – данные не сохраняются между запусками.
